# ブラックサンダー / Black Thunder ⚡🍫

コーディングのお供に。ファイルを保存したりコマンドを実行したりするたびに、
ブラックサンダーがザクッとご褒美をくれる、おやつ系 VSCode 拡張機能です。

> ※ 非公式のファン作品です。「ブラックサンダー」は有楽製菓株式会社の商品・商標です。
> 本拡張機能は同社とは関係ありません。配布の際は商標・画像の扱いにご注意ください。

## 機能

- **ご褒美ポップアップ** — 保存時やコマンド実行時に、雷級のうまさをたたえるメッセージを表示。
- **テスト成功ご褒美** — テスト系タスク（group=test、または名前に test/spec/jest/vitest/pytest を含む）が終了コード 0 で終わったら自動でご褒美。
- **ガチャ要素** — ご褒美のたびに重み付き抽選。`N ブラックサンダー / R ビッグサンダー / SR 白いブラックサンダー / UR 金のブラックサンダー` が出現し、レア以上は派手にお祝い。引いた累計はコレクションに記録。
- **ステータスバー表示** — `⚡ 3` のように、今日食べた個数をカウント（日付が変わると自動リセット）。
- **週間グラフ** — サイドバーに直近 7 日間の個数を棒グラフ表示。
- **サウンド再生** — ご褒美のたびに、生成した雷の効果音を鳴らす（OS 標準プレーヤーを使用）。
- **サイドバー** — アクティビティバーの ⚡ アイコンから、キャッチコピー・ガチャ結果・週間グラフ・コレクションと「ザクッと食べる」ボタンを表示。

## コマンド

| コマンド | 説明 |
| --- | --- |
| `Black Thunder: ブラックサンダーを食べる` | ご褒美をもらってカウント +1 |
| `Black Thunder: ご褒美をもらう` | 同上 |
| `Black Thunder: 今日のカウントをリセット` | 今日の個数を 0 に戻す |

## 設定

| 設定キー | 既定値 | 説明 |
| --- | --- | --- |
| `blackthunder.rewardOnSave` | `true` | 保存時にご褒美を出すか |
| `blackthunder.rewardOnTestPass` | `true` | テスト成功時にご褒美を出すか |
| `blackthunder.enableSound` | `true` | 効果音を鳴らすか |
| `blackthunder.saveRewardCooldownSeconds` | `60` | 保存ご褒美の連発を防ぐクールダウン秒数 |

## 開発・実行方法

```bash
npm install
npm run watch        # 別ターミナルで watch ビルド
```

VSCode でこのフォルダを開き、`F5`（Run Extension）で拡張機能ホストを起動して動作確認できます。

### パッケージ化（.vsix の作成）

```bash
npm install -g @vscode/vsce
vsce package
```

> `package.json` の `publisher` を自分の Publisher ID に書き換えてからパッケージ化してください。
> マーケットプレイスのアイコン `media/icon.png` は仮のものなので、必要に応じて差し替えてください。
